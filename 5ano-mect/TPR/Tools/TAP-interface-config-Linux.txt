Creation of TAP interface in Linux

To create a TAP interface in Linux:

ip tuntap add dev tap0 mode tap
ip link set up dev tap0
ip addr add 10.0.1.100/24 dev tap0

# To define that all networks 10.0.0.0/8 are accessible via 10.0.1.1
ip route add 10.0.0.0/8 via 10.0.1.1
