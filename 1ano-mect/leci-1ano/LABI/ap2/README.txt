#REGRAS PARA USAR ESTE REPOSITORIO

- git log
	(HEAD -> master) esse é o ponto onde tu estás
	(origin -> master) esse é o ponto onde o repositorio remoto está
	Informacao dos commits que foram feitos

- git add
	usar "git add <ficheiro>" e nao "." para nao adicionar ficheiros desnecessarios

- git push --all
	mandar a informacao que escreve-mos/altera-mos para o repositorio

- git pull origin master
	usar este comando antes de comecarmos a trabalhar para ter a ultima versao no repo local

- git clone <adress>
	usar este comando quando já ha um repos remoto e queremos ter uma copia no repos local
